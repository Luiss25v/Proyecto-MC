# Proyecto MC · Ayuda-Memoria Interactiva (Funciones)

Web interactiva que permite:
- **Lógica proposicional:** tabla de verdad + código MATLAB
- **Conjuntos:** operaciones + Venn + código MATLAB
- **Funciones:** gráfica en canvas + script MATLAB

## Ejecutar local
Abre `index.html` en tu navegador (doble clic).

> Recomendado: usar VS Code + extensión “Live Server” para recargar automático.

## Publicar en GitHub Pages
1. Crea un repo en GitHub (ej: `proyecto-mc-funciones`)
2. Sube: `index.html`, `styles.css`, `app.js`, `README.md`
3. Ve a **Settings → Pages**
4. Build: **Deploy from a branch**
5. Branch: `main`, folder: `/root`
6. Guarda y entra al link que GitHub te da.
